Crypto Collective Services

This hack predicts cryptocurrency trends based on live tweets on Twitter. This data is then processed Micrsoft Azure Text Analytics API which evalutes users' sentiment concerning cryptocurrency prospects. This in turn is converted to percentage numbers which determines if the tweet is positive or negative in tone. This data advises users in real time about the state of different cyrptocurrencies.
